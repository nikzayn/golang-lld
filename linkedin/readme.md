# Designing LinkedIn
- In this design, we will be designing famous social media platform for working professional and people who build networking connection to generate more good opportunies and jobs.

# Design Approach
- We will design using bottom-up approach
- We need to identify the smallest components first like to like a post and add comment
- Then, we move forward with to create pages, groups and profile.
- Rpeeat the steps till we cover all the features to design LinkedIn platform.
